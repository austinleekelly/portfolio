## 📌 Summary

_The LAN Security Using Switch Features course provides essential knowledge for securing Local Area Networks (LANs) by utilizing switch-level features. Topics include managing MAC addresses, implementing port security, defending against common network attacks, and enhancing security in VLANs._

## 📂 Certification Overview

- **🏆 Certification:** LAN Security Using Switch Features
    
- **🎓 Issuing Organization:** [Department of Homeland Security](https://www.dhs.gov/)
    
- **📆 Date Earned:** 12/28/2017
    
- **📜 Certification Level:** Intermediate
    

## 🎯 Key Learning Outcomes

- **LAN Security Vulnerabilities** – _Identify potential security risks in LAN configurations and best practices to mitigate them._
    
- **Traffic and Host Control** – _Understand the methods to control traffic flows and monitor host security health on LANs._
    
- **Defending Network Protocols** – _Learn to defend against attacks targeting STP, VLAN, and VTP configurations._
    

## 🚀 Skills Acquired

### 🛠 **Core Skills**

- **Technical Skills:** VLAN security, MAC & IP spoofing prevention, traffic flow control, host health monitoring
    
- **Soft Skills:** Network security analysis, risk management, decision-making in security configurations
    

### 💻 **Tools & Technologies**

- **Switch Features** – _MAC address filtering, port security, authentication, VLAN configurations_
    
- **Network Access Control (NAC)** – _Frameworks for secure network access and connection point management_
