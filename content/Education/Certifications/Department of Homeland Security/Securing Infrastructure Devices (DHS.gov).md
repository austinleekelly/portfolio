## 📌 Summary

_Securing Network Infrastructure Devices course provides an in-depth overview of securing the critical components of network infrastructure, such as routers, firewalls, switches, and servers. It covers strategies for safeguarding these devices, including network segmentation, access control, and the implementation of robust security practices to mitigate risks from cyber threats._

## 📂 Certification Overview

- **🏆 Certification:** Securing Network Infrastructure Devices
    
- **🎓 Issuing Organization:** [Department of Homeland Security](https://www.dhs.gov/)
    
- **📆 Date Earned:** 12/29/2017
    
- **📜 Certification Level:** Intermediate
    

## 🎯 Key Learning Outcomes

- **Network Segmentation and Isolation** - _Understand how proper segmentation and segregation of networks can prevent lateral movement of cyber threats and contain security breaches._
    
- **Access Control and Authentication** - _Learn how to implement secure access controls and multi-factor authentication to prevent unauthorized access to network infrastructure devices._
    
- **Device Hardening and Configuration** - _Explore methods to secure network devices by disabling unnecessary services, using strong encryption, and implementing robust password policies._
    
- **Out-of-Band Management** - _Understand the benefits of using alternative communication paths for secure network management that limits exposure to attackers._
    
- **Supply Chain Integrity** - _Learn how to validate hardware and software integrity to prevent the use of counterfeit or compromised network devices._
    

## 🚀 Skills Acquired

### 🛠 **Core Skills**

- **Technical Skills:** Network infrastructure security, device hardening, segmentation strategies, access control, multi-factor authentication
    
- **Soft Skills:** Analytical thinking, security strategy planning, risk assessment

- **Network Segmentation Technologies** - _Private VLANs, VPNs, Virtual Routing and Forwarding (VRF)_
    
- **Network Management Protocols** - _SNMPv3, HTTPS, SSH, AAA Services_
