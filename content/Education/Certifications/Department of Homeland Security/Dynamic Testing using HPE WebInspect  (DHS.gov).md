### 📌 Summary

_This course introduces the use of dynamic application security testing (DAST) tools, specifically HPE WebInspect, to identify and manage web application vulnerabilities. It provides foundational skills in using automated testing tools to evaluate and mitigate security risks, relevant to both developers and cybersecurity professionals._

## 🎯 Key Learning Outcomes

- Identify tools for dynamic security testing of web applications
    
- Learn how to use HPE WebInspect for vulnerability detection
    
- Understand testing results from both development and cybersecurity perspectives
