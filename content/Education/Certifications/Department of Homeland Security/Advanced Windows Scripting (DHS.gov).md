## 📌 Summary

_Advanced Windows Scripting equips learners with practical skills in automating Windows tasks using BATCH and Visual Basic Scripting (VBS), including integration with Windows APIs._

## 📂 Certification Overview

- **🏆 Certification:** Advanced Windows Scripting
    
- **🎓 Issuing Organization:** [Department of Homeland Security](https://www.dhs.gov/)
    
- **📆 Date Earned:** 12/28/2017
    
- **📜 Certification Level:** Basic
    

## 🎯 Key Learning Outcomes

- **Visual Basic Scripting Fundamentals** - _Learn the structure and syntax of VBS for automation._
    
- **Command-Line Automation** - _Use redirection, piping, and multi-command operations._
    
- **API Integration** - _Access Windows system functionality from scripts._
    

## 📘 Curriculum Overview

Focuses on BATCH scripting, Visual Basic Scripting, and integrating both for more powerful system administration. Includes demonstrations and a final assessment.
