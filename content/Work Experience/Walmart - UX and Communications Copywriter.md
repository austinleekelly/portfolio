## 📌 Summary

At Walmart, I draft, edit, review, and govern content that enables support for drivers and agents across Last Mile Delivery, including agent workflows. In parallel, I build AI tools on Code Puppy that have measurably reduced documentation time, expanded content access across the org, and automated compliance review. I work at the intersection of content systems, agentic AI, and operational workflows for a team that touches every corner of the LMD business.

---

## 📂 Role Overview

- **💼 Company:** Walmart
- **🏢 Team:** Driver Engagement and Communications, Last Mile Delivery
- **📆 Duration:** July 2025 - Present
- **🏢 Employment Type:** Full-time
- **📍 Location:** Bentonville, Arkansas

---

## 🎯 Key Responsibilities

- **Content Authoring and Governance** - Draft, edit, review, and govern content for driver and agent support, including agent-facing workflows that guide how support agents respond to Spark Driver issues.

- **Agent Workflow Management** - Maintain and improve agent workflows in a platform used by contact center agents supporting Spark Drivers across the country.

- **AI Tool Development** - Design and build AI-powered tools on Code Puppy using vibe coding and the Anthropic API, creating internal utilities that reduce manual effort and improve content accuracy.

- **Cross-Functional Collaboration** - Work across Last Mile Delivery, legal, compliance, and vendor teams to ensure content accuracy, consistency, and readiness for production.

- **Knowledge System Design** - Build tools and workflows that make content more accessible, auditable, and reusable across the Driver Enablement team and broader LMD stakeholders.

---

## 🚀 Achievements and Impact

- ✅ **Reduced agent workflow documentation time from ~3 hours to ~3 minutes** by building a JSON to Word converter that auto-generates human-readable exports from the agent workflow platform.

- ✅ **Expanded agent workflow access across LMD** with Flowbase Lite, a read-only viewer for ~200 agent workflows previously inaccessible to stakeholders without platform credentials.

- ✅ **Automated compliance review** with DEKU, a content cross-referencing agent built on Code Puppy that checks for language discrepancies, logic conflicts, and legal and compliance risk.

- ✅ **Eliminated vendor dependency for PDF exports** by building CAPSULE, a browser-based tool that lets any team member bulk download Spark Driver Resource Center content previews on demand.

- ✅ **Built a legal-ready redline workflow** that compares draft and production agent workflow JSONs and generates a formatted change document for legal review.

- ✅ **Improved rich text editing for legacy content** with a local-first HTML editor that allows previewing and loading legacy HTML into the agent workflow platform.

- ✅ **Self-initiated all tools without being asked**, building them in parallel with core responsibilities to solve real team problems.
